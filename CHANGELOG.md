## [1.1.0] - 2023-09-09

### Added
- Viewer column visibility option.
- Viewer reload function.
- Viewer output csv function.
- Collect and show windows device information.
    - ComputerManufacturer
    - ComputerModel
    - Processor
    - BIOSManufacturer
    - BIOSVersion
    - BitLockerStatus

### Changed
- Change Viewer UI.
- Change database access logic.
- Refactoring.

## [1.0.0] - 2023-08-27

### Added

    - HostName
    - UserName
    - OSName
    - OSBuildNumber
    - OSVersion
